# 🎓 Teste Front-End – Módulo Academia Dev

Projeto desenvolvido como exercício de Front-End para simular a interface de um módulo de curso da **Academia Dev**.

O objetivo foi construir uma página responsiva utilizando **HTML, CSS e JavaScript puro**, sem uso de frameworks, aplicando boas práticas de layout moderno com **Flexbox** e manipulação de DOM.

---

# 📸 Preview

Interface composta por:

- Card principal com destaque do módulo
- Descrição do conteúdo do curso
- Carrossel de aulas complementares
- Layout responsivo

---

# 🚀 Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Flexbox**
- **Google Fonts**

---

# 🧠 Funcionalidades implementadas

✔ Layout responsivo  
✔ Estrutura semântica em HTML  
✔ Carrossel funcional em JavaScript puro  
✔ Navegação com botões anterior/próximo  
✔ Recalculo automático do carrossel ao redimensionar a tela  
✔ Uso de variáveis CSS para padronização de cores e fontes
